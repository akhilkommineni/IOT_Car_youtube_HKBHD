---Board Manager link for NodeMCU

http://arduino.esp8266.com/stable/package_esp8266com_index.json
How to : 
1. paste this link in "Arduino IDE --> File/Preferences/Additional Boards Manager URLs/"
2. After pasting goto "Arduino IDE --> Tools/Board/Board Manager/"
3. Search for esp8266(Version used in the youtube video is 2.4.1) select the version and click install
